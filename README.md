# MSPaint-Any-Polygones-2020

## Overview
**MSPaint-Any-Polygones-2020** is a program that enhances Microsoft Paint by allowing users to create polygons with any number of sides and customizable sizes through a GUI. This functionality is not natively available in MS Paint.


## What I Learned
- **`Tkinter` GUI library**: Developing user-friendly interfaces.
- **Geometry and Algebra**: Implementing mathematical formulas to dynamically generate polygons.
